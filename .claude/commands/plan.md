# Plan

Break work into actionable tasks.

## What to Do

1. List all tasks needed
2. Order by dependencies
3. Estimate complexity (simple/medium/complex)
4. Add to TodoWrite
5. Get user approval

## Example Output

```
Implementation Plan: [Feature]

Tasks:
1. [Task 1] - Simple
   - What: [description]
   - Files: [affected files]

2. [Task 2] - Medium
   - What: [description]
   - Depends on: Task 1

3. [Task 3] - Complex
   - What: [description]
   - Verification: [how to test]

Total: 3 tasks

Does this plan look right?
```

## When to Use

- After brainstorming
- Before implementation
- Multi-step tasks
- Complex features
